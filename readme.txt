== Description ==

The easiest way to add a transparent header to the Beaver Builder theme.

### Features
* Easily display login/logout link on a menu
* Simple dropdown in the menu to choose the location

== Installation ==

### Plugin Installation
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Can I use this with another theme? =

No. It only works with the Beaver Builder theme

= Can I use it with a child theme? =

Yes. As long as it is a child theme of the Beaver Builder theme
